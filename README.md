# Система учёта заявок книжного магазина «Бакен»

Веб-приложение для управления заявками и задачами в книжном магазине. Позволяет клиентам создавать заявки, отслеживать их статус, а менеджерам — назначать исполнителей, изменять статусы и контролировать выполнение.

## 🚀 Функционал

- 🔐 **Авторизация и роли** (клиент, менеджер, администратор)
- 📝 **Создание заявок** (тема, описание, категория)
- 📋 **Просмотр списка заявок** с фильтрацией и поиском
- 📄 **Детальная карточка заявки** с комментариями и историей изменений
- 👥 **Назначение исполнителей** (только для менеджеров)
- 🔄 **Изменение статусов заявок** (новая, в работе, выполнена, отклонена)
- 📊 **Панель управления менеджера** со статистикой
- 💬 **Комментарии и история изменений**
- 📱 **Адаптивный дизайн** (Bootstrap 5)

## 🛠 Технологии

- **Backend:** Django 5.1.6
- **Frontend:** Bootstrap 5, HTML, CSS
- **База данных:** PostgreSQL 15
- **Контейнеризация:** Docker, Docker Compose
- **Веб-сервер:** Gunicorn
- **Дополнительно:** django-filter, django-crispy-forms

## 🏃‍♂️ Запуск проекта

### Локальный запуск (без Docker)

```bash
# 1. Перейдите в папку с проектом
cd hab/nl

# 2. Создайте и активируйте виртуальное окружение
python -m venv venv
source venv/bin/activate      # Linux/Mac
venv\Scripts\activate         # Windows

# 3. Установите зависимости
pip install -r requirements.txt

# 4. Примените миграции
python manage.py migrate

# 5. Создайте суперпользователя
python manage.py createsuperuser

# 6. Запустите сервер
python manage.py runserver

Запуск через Docker (рекомендуемый)
# 1. Перейдите в корневую папку проекта
cd hab

# 2. Соберите и запустите контейнеры
docker compose up --build

# 3. Для остановки
docker compose down
